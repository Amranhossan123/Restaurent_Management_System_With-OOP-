# Restaurent_Management_System_With-OOP

A simple and efficient Restaurant Management System built with Python using **Object-Oriented Programming (OOP)** logic.

##  How to Play / Run
1. Navigate to the project folder.
2. Run the main script:
   ```bash
   python main.py

## System Features

* Menu Management: View and manage food items and prices.

* Order System: Take and process customer orders.

* Billing: Automatic total bill calculation.

* OOP Structure: Uses Classes and Objects for clean code.

## Tech Stack
* Language: Python 3.x

* Modules: Multiple .py files for modular design.

* Logic: Functional and Object-Oriented approach.
